## 欢迎 🎉

欢迎来到我的个人博客！这是我分享技术、思考和成长的空间。

## 博客特色

这个博客采用了现代化的技术栈和精心设计的功能：

### 技术特性

- **🚀 极速加载**：基于 Astro 构建，静态生成，加载速度极快
- **🎨 深色模式**：支持浅色/深色主题切换
- **📱 响应式设计**：完美适配各种设备
- **♿ 无障碍访问**：遵循 WCAG 标准

### SEO 优化

博客内置了完整的 SEO 优化：

- 自动生成 meta 标签
- Open Graph 和 Twitter Card 支持
- 自动生成 sitemap.xml 和 robots.txt
- JSON-LD 结构化数据

### 内容组织

- **标签系统**：通过标签快速找到相关内容
- **分类系统**：按主题浏览文章
- **RSS 订阅**：支持通过 RSS 阅读器订阅

### 互动功能

- **打赏支持**：如果文章对你有帮助，可以通过打赏支持我
- **社交媒体**：在多个平台找到我

## 代码示例

博客支持代码高亮，例如：

```javascript
// 一个简单的 JavaScript 函数
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

```python
# Python 示例
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
